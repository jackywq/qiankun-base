# 一个基于vue2.x基座搭建的qiankun微前端demo
 - 主应用vue2.x
 - 子应用分别为vue和react



## 下载命令

```javascript
git clone https://github.com/jackywq/qiankun-base.git
```



## 启动

### qiankun-base基座应用
```
npm install
npm run serve
```
<span style="color:red">【注意】</span> 需要同时启动 **qiankun-react**  &  **qiankun-vue** 两个子应用才能实现预期效果

#### qiankun-react子应用

```
npm install 
npm run start
```

#### qiankun-vue子应用

```
npm install
npm run serve
```
